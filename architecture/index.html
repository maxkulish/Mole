What is Mole?

Mole is an all-in-one macOS system optimization and disk analysis tool written in Go with shell script support. It provides an interactive terminal UI replacement for tools like CleanMyMac, AppCleaner, DaisyDisk, and iStat Menus.

It has two main commands: mo analyze - an interactive terminal UI for visualizing disk usage, finding large files, and safely deleting them via macOS Trash; and mo status - a real-time system health dashboard showing CPU, GPU, memory, disk, network, and battery stats with an overall health score.

All operations are local-only. The tool reads filesystem metadata, runs macOS system commands (pmset, system_profiler, mdfind, du), and writes only to ~/.config/mole/ and ~/.local/share/mole/. No data ever leaves the machine during normal operation.
