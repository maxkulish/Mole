What This Application Does NOT Do
  • Make network calls from Go code (zero HTTP/HTTPS in Go binaries)
  • Collect or transmit user data to any third party
  • Use telemetry, analytics, or crash reporting
  • Access the clipboard, keychain, or credential store
  • Use unsafe Go code (zero unsafe package usage)
  • Permanently delete files (all deletions go to macOS Trash)
  • Listen on network ports or act as a server
  • Install background services, LaunchAgents, or daemons
  • Require root / administrator privileges in Go binaries
  • Modify system files (read-only access to system paths)
  • Auto-run scheduled cleanups without user initiation
  • Sync data to the cloud or any external service
